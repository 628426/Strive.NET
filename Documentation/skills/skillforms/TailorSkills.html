Dye                     Tailors can make a variety of wonderous 
                        creations.

Duration                N/A
Use                     Can add colour to clothing.
Syntax                  Dye (appropriate item) (colour choice)
Affect                  If the target item is made of cloth or anything 
                        that can be dyed, its short and long descs are 
                        changed to be in the desired colour.
        
                        If you fail in dyeing an item of clothing it 
                        could end up a random colour.

Skills it counteracts           None
Skills it compliments           None
Level restrictions              None
Major element                   None
Prerequisites                   None

Practice Cost                   Can practice to 75%, costs 1 practice point 
                                for every percent. Max % is 95.

Casting costs etc               25 energy
Required Components             None
Starts Combat?                  No
Used During Combat?             No

Remake                  Renames an item of clothing

Duration                N/A
Use                     See above
Syntax                  Remake (clothing item) (type of desc) (desc)
Affect                  Can rename basically any item made of 
                        appropriate materials i.e.

                        Remake (vest) long (desc) changes the long 
                        description

                        Remake (vest) short (desc) changes the short 
                        description

                        Remake (vest) desc enters editor to change 
                        look at description (ie extra desc)

                        The (vest) argument would have to be one of
                        the keywords in the objects name for the skill
                        to be successful.

Use                     See Above
