Write                   You can write on parchment objects

Duration                N/A
Use                     See above
Syntax                  Write (object)
Affect                  Allows you to modify the extra description of 
                        a parchment object to put text onto it so 
                        that when that object is read they see what 
                        you wrote on the parchment. If you fail your 
                        write skill the parchment is destroyed.

Skills it counteracts           None
Skills it compliments           Seal Letter
Level restrictions              None
Major element                   None
Prerequisites                   None

Practice Cost                   Can practice to 75%, costs 1 practice 
                                point for every percent. Max % is 95.

Casting costs etc               10 energy
Required Components             Parchment, quill object held
Starts Combat?                  No
Used During Combat?             No

                        durations of spells targeting you.

Research                Can learn information about objects and mobs 
                        and rooms. You have to be in a special
                        library room (there will be one in each
                        major town or city) to use the skill.

Duration                N/A
Use                     To help learn something about people, places 
                        and things you know of.

Syntax                  Research (target area, object or mob)
Affect                  What research tells you will depend upon what 
                        it is you are researching.

                        Research (area)
                        Does research on the target area, this will 
                        tell you some details about the area such as
                        the level of the mobs that are in the area.
                        Maybe a shopkeeper list too.

                        Research (mob)
                        This will tell you the level of that mob as 
                        well as that mobs class, race and size.

                        Research (object)
                        This will tell you the value of that object 
                        and what type of object it is. It will also 
                        tell you the material that object is made 
                        out of.

                        Perhaps we will need a no_research flag on mobs, 
                        objects and rooms, though perhaps we will not 
                        need it.

Skills it counteracts           None
Skills it compliments           None
Level restrictions              None
Major element                   None
Prerequisites                   None

Practice Cost                   Can practice to 75%, costs 1 practice 
                                point for every percent. Max % is 95.

Casting costs etc               None
Required Components             None
Starts Combat?                  No
Used During Combat?             No

Feign Death             Allows the student to act as though they are 
                        already dead to avoid confrontation

Duration                N/A
Use                     See above
Syntax                  Feign Death
Affect                  In order for this to be useful we will need 
                        to change the way in which corpses are 
                        displayed. If the dead creature is an animal 
                        you see the corpse of an animal. If it is a 
                        certain race you see the corpse of that race 
                        so that when you feign death you become a 
                        corpse of the race you are. 

                        With this in mind feigning death will allow 
                        you to appear to be a corpse. Your short desc 
                        will change to "the corpse of a (race) lies here"
                        and your look at desc will be the same. Whilst 
                        you are feigning death you are aware of all that 
                        goes on in the room around you. As soon as you 
                        type a command like say or get or try to move 
                        you stop feigning death. You cannot be attacked 
                        while you are feigning death until revealed by 
                        the check pulse skill.

Skills it counteracts           None
Skills it compliments           None
Level restrictions              None
Major element                   None
Prerequisites                   None
Practice Cost                   Can practice to 75%, costs 1 practice 
                                point for every percent. Max % is 95.

Casting costs etc               None
Required Components             None
Starts Combat?                  No
Used During Combat?             No
