unarmed combat          When a person using no wielded weapon is 
                        fighting they gain an advantage to their 
                        chance to hit.

Duration                N/A
Use                     To allow people to train in the use of 
                        weapons and make sure that only the people 
                        who should be able to realistically use a 
                        weapon can use it.

Syntax                  None
Affect                  Whilst wielding no weapon the person gets 
                        double their existing hit roll, so twice as 
                        many tries to try and hit an opponent if the 
                        skill is used successfully.

Skills it counteracts           None
Skills it compliments           None
Level restrictions              None
Major element                   None
Prerequisites                   None

Practice Cost                   Can practice to 75%, costs 1 practice 
                                point for every percent. Max % is 95.

Casting costs etc               None
Required Components             None
Starts Combat?                  No
Used During Combat?             Yes

Work                    Makes money     

Duration                N/A
Use                     Allows the player to make some money by 
                        working
Syntax                  Work
Affect                  If you are in a city square you receive 2 
                        gold coins for every tick (30 seconds?) spent 
                        working. By typing work you enter working 
                        mode, and your pemote is changed to reflect 
                        the fact you are working. Using the say 
                        command or trying to move etc takes you out 
                        of work mode. You can only work while you have 
                        the energy to work. Whilst working you lose 20 
                        energy per tick and do not regain energy each 
                        tick you are working.

                        This works out that a level 1 person with 500 energy 
                        can earn 50 gold coins in 25 ticks (just over 10 mins) 
                        before resting.

Skills it counteracts           None
Skills it compliments           None
Level restrictions              None
Major element                   None
Prerequisites                   None

Practice Cost                   Can practice to 75%, costs 1 practice point 
                                for every percent. Max % is 95.

Casting costs etc               20 energy per tick
Required Components             None
Starts Combat?                  No
Used During Combat?             No
