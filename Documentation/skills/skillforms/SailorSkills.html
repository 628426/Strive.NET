Signaling               A brief set of signals using hand signs from 
                        1 room away

Duration                N/A
Use                     To inform others of certain things that are 
                        happening
Syntax                  Signal (direction) (command)
Affect                  We will need to come up with a list of signals 
                        we think appropriate and useful to the players. 
                        If you fail your signal the message you send 
                        in the direction you chose is nonsensical. 
                        Anyone in the target room with the signal skill 
                        can interpret a successful signal.

Skills it counteracts           None
Skills it compliments           None
Level restrictions              None
Major element                   None
Prerequisites                   None
Practice Cost                   Can practice to 75%, costs 1 practice 
                                point for every percent. Max % is 95.

Casting costs etc               None
Required Components             None
Starts Combat?                  No
Used During Combat?             Yes

Hire                    Can hire crewman to accompany the sailor.

Duration                N/A
Use                     See above
Syntax                  Hire (crewman)
Affect                  Cremen are spec sailor mobs and have a 
                        level equal to the sailor. A sailor can 
                        hire 1 creman for every 25 levels he or she 
                        has. Sailors can group with crewmen and 
                        order them to do some basic commands. Hiring 
                        crewmen of course costs money, 1 gold per 
                        crewman per hour. Crewmen leave if they 
                        cannot be paid.
