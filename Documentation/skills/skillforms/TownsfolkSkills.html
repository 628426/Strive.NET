Seal Letter             Create a unique seal on a letter object based on 
                        the players name. 

Duration                N/A
Use                     So that letters sent have a unique seal, you know 
                        that if you see a certain seal the letter has 
                        remained unopened and is from that player.

Syntax                  Seal (appropriate letter object)
                        Seal (with no appropriate letter object sets the 
                        text to the seal description)
                        Open (sealed letter)

Affect                  The letter object will have its short, long and 
                        look at descriptions change based upon the players 
                        name and seal type.

                        i.e. Seal (Jeweled Sword)
                             Seal (Letter)
                        Will change the letter object to the following:
                        Long description: A sealed letter is here.
                        Short description: a sealed letter
                        Look at description: A letter sealed with the 
                        Jeweled Sword of Isrejin.
                        Breaking the seal returns the original descriptions 
                        to the object.                          

Skills it counteracts           None, you can still break a sealed letter 
                                and read it if you intercept it. But the 
                                seal is broken.

Skills it compliments           None really
Level restrictions              None
Major element                   N/A
Prerequisites                   None
Practice Cost                   1 percentage per practice point spent. 
                                Can practice all the way up to 100%

Casting costs etc               None
Required Components             You need an appropriate letter object to be 
                                able to seal it.
Starts Combat?                  No
Used During Combat?             No

Swim                    Obviously not everyone has to train it, and it 
                        will only entitle people to swim in shallow water I 
                        should think. We certainly don’t want people just 
                        swimming across oceans.

Duration                It affects you in permanence
Use                     To move over certain water squares.

Syntax                  It’s a skill that is always in affect, so you don't 
                        initiate it with a command.

Affect                  Will be able to walk over and into squares that are 
                        flagged water_surface. So they won't be able to swim 
                        in squares flagged deep_water or underwater. This is 
                        something we will need to tell the builders.

Skills it counteracts           None
Skills it compliments           Skills you are required to be in water rooms 
                                to perform.

Level restrictions              None
Major element                   None
Prerequisites                   None

Practice Cost                   1 percentage per practice point spent. Can 
                                practice only to 75% have to learn the rest up 
                                to 95% max

Casting costs etc               Costs double your energy to move using the swim 
                                skill (ie in water)

Required Components             None
Starts Combat?                  No
Used During Combat?             Yes, to flee a fight in shallow water. Note that 
                                all fights conducted in water_surface should have 
                                severe restrictions on to hit and damage.

                        others to find.

Use                     Remove corpses from cluttering up the room

                        for quite a long time until it decays.

Build Fire              Anyone should be able to build a fire so long 
                        as they have a flint and tinderbox. Fires light 
                        up darkened rooms and can only be built outdoors. 
                        They cannot be built when the weather is rainy. 
                        They enable you to sleep more comfortably or at 
                        all.

Duration                Dependent on weather, and raw materials used in 
                        the fire.

                        If its rainy any fire you build will last for half
			the duration listed below.
                        If its overcast your fire duration is 3/4
                        If its clear your fire duration is normal
                        
                        For each piece of lumber used in the fire to 
                        create it the fire stays for 2 hours based upon 
                        the current game time we have. If our game 
                        time changes so will this have to.

                        If no lumber is used to start the fire then it lasts
                        for 1 hour, explained by the gathering of twigs / 
                        branches etc.

Syntax                  Build Fire 

Affect                  Creates a fire object in the room:
                        If 0-1 piece of lumber spent the fire is a small fire
                        If 2 -4 pieces of lumber spent the fire is a 
                        If 4-6 pieces of lumber spent the fire is a large fire
                        > 8 pieces spent the fire is a roaring fire

                        If you use the build fire command in a room where 
                        a fire already exists it will add the lumber you enter 
                        to the current fire.

                        What I would also like to propose is that we have a 
                        temperature system associated with our weather system. 
                        If the weather is cold and there is no fire built, 
                        then players should not be able to sleep outdoors.

                        The way I would like fires to work in terms of healing
                        rates is as follows:

                        Healing rate sleeping indoors = normal
                        Healing rate sleeping indoors with a fire= +50%
                        Healing rate sleeping outdoors = half
                        Healing rate sleeping outdoors with a fire = normal
			Healing rate sleeping outdoors in rain 25% or normal
                        

Skills it counteracts           None
Skills it compliments           The labourer chop lumber skill and the cook 
                                skill

Level restrictions              None
Major element                   N/A
Prerequisites                   N/A

Practice Cost                   1 practice point (pp) per %, can prac to 75% 
                                with a learned max of 95%       

Casting costs etc               Costs 50 energy to build a fire
Required Components             Flint and tinder and some form of lumber.
Starts Combat?                  No
Used During Combat?             No

Combat Basics           This allows you to use the stance command. You must 
                        train each stance individually however.

Use                     To give you the edge and to provide variation in 
                        combat by allowing you to use stances.

Syntax                  stance (stance type)
Affect                  You can use the stance command. If you fail you 
                        just fail to assume that stance for fighting and 
                        retain your current stance. This would only have 
                        an affect if you want to try and change your 
                        stance whilst fighting using a more powerful skill.

                        Note also that this skill gives you access to use
                        the passive and normal stances.

Skills it counteracts           None
Skills it compliments           None
Level restrictions              None
Major element                   None
Prerequisites                   None
Practice Cost                   Can practice to 95%, costs 1 practice point 
                                for every percent. 

Casting costs etc               None

Starts Combat?                  No

                                for combat.
