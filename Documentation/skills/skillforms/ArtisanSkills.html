Inset Gem

                Certain gems add value or affects to items. Items should 
                have a realistic number of gem slots for gems to be inset.
                e.g. a breastplate inset with diamond would have an 
                increased value and ac. You need to have the gems of
                course :)

Devaluate

                Can add different gems into an item without being noticed, 
                which have negative affects. For example insetting glass
                into a breasplate would reduce its ac and increase its
                chance of being damaged or destroyed. Devaluate just
                allows these materials to be inset without being visibly
                obvious.
