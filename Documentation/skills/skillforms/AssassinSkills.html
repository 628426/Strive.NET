                to the combine skill and allow the assassin to mix various 
                poisons. Perhaps we need to implement a system like DR 
                hereby assassins have to train each poison they can make 
                serperately. The most powerful and deadly poisons are
                created by alchemists however, even though an assassins
                poisons are pretty potent.
