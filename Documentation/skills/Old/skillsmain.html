This page is designed to break up the rather lengthy proposed skill list into more manageable sections which are easier to understand. This page is the outermost page, and will help you to navigate through the specialisation tree to choose specific specialisations to browse through the skills of.

You should only check boxes on this page if you are happy with ALL skills for that specialisation. You may choose as many specialisations as you beleive are complete on this page and submit this to me via email.

Comments posted using the submission form at the bottom of this page should be overall comments about possible changes. Specific comments on a specialisation should be submitted via the submission page for that specialisation. Be sure to include your Immortal Name in the submission or I won't know who sent it!

When all imms have submitted their feedback I will make the results available.

Before looking into the specialisations, you should probably read the guidelines. There is one other page you should know about and that is the extras page which contains extra details not mentioned specifically on the specialisation pages.
